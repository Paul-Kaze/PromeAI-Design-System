# Inspira — Canvas App UI Kit

A click-thru recreation of Inspira's core workspace: Home → Canvas + Chat. Built off the `promeai-inspira-web` codebase and the Figma chat-history / toast frames.

## Screens

- **Home** — `Hello, Inspira!` greeting + grid of canvas cards. Click any card (or "New Canvas") to enter the workspace.
- **Workspace** — three-pane: left chat-history sidebar, center canvas, right composer.
- **Toast stack** — success / error variants triggered on save, delete, out-of-credits.

## Components

| File | Role |
|---|---|
| `Header.jsx` | Fixed 60px top bar with logo, canvas title, coin pill, avatar menu |
| `Sidebar.jsx` | Collapsible chat-history list with "New Chat" + project group |
| `HomeGrid.jsx` | "Hello, Inspira!" hero + canvas cards grid |
| `CanvasStage.jsx` | Mock excalidraw surface + floating tool rail + popbar |
| `Composer.jsx` | Prompt textarea + model chip + image-quantity + send |
| `ChatMessage.jsx` | Assistant + user message bubbles with generated-image grid |
| `Toast.jsx` | Success / error toasts |
| `Menu.jsx` | Dropdown menu used on canvas title, avatar, model selector |
| `Button.jsx` | Primary gradient / secondary / white / ghost variants |
| `Icon.jsx` | Lucide stand-ins (see README caveat) |

## Notes

- Icons: Lucide stand-ins — the real product uses an iconfont sprite generated by `src/scripts/font-builder/`. Same stroke weight, same container size; swap when lifting into production.
- "Jaaz" strings seen in the locale files are rewritten to "Inspira" here.
- State is in-memory; refresh resets the demo.
